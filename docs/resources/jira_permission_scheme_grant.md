# atlassian_jira_permission_scheme_grant (Resource)

Manages a permission grant within a Jira Cloud permission scheme. Grants are immutable; any change forces recreation.

## Example Usage

```terraform
resource "atlassian_jira_permission_scheme_grant" "example" {
  scheme_id        = atlassian_jira_permission_scheme.example.id
  permission       = "BROWSE_PROJECTS"
  holder_type      = "group"
  holder_parameter = atlassian_jira_group.example.name
}
```

## Schema

### Required

- `holder_type` (String) The type of the permission holder.

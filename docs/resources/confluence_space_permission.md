# atlassian_confluence_space_permission (Resource)

Manages a Confluence Cloud space permission. All attributes except id are ForceNew.

## Example Usage

```terraform
resource "atlassian_confluence_space_permission" "example" {
  space_key        = atlassian_confluence_space.example.key
  space_id         = atlassian_confluence_space.example.id
  principal_type   = "group"
  principal_id     = atlassian_jira_group.example.group_id
  operation_key    = "read"
  operation_target = "space"
}
```

### Required

- `operation_key` (String) The permission operation key (e.g. "read", "create", "delete"). Changing this forces recreation.
- `operation_target` (String) The operation target (e.g. "space", "page", "comment"). Changing this forces recreation.

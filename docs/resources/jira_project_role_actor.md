# atlassian_jira_project_role_actor (Resource)

Assigns a user or group to a project role in a specific Jira Cloud project. This resource is immutable; any change forces recreation.

## Example Usage

```terraform
resource "atlassian_jira_project_role_actor" "example" {
  project_key = atlassian_jira_project.example.key
  role_id     = atlassian_jira_project_role.example.id
  actor_type  = "atlassianGroup"
  actor_value = atlassian_jira_group.example.name
}
```

## Schema

### Required

- `actor_type` (String) The type of actor: atlassianUser or atlassianGroup.
- `actor_value` (String) The actor value: accountId for users, group name for groups.
- `project_key` (String) The key of the project.
- `role_id` (String) The ID of the project role.

- `id` (String) Composite ID: {project_key}/{role_id}/{actor_type}/{actor_value}.
